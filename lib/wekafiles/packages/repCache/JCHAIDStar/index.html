JCHAIDStar: Class for generating a decision tree based on the CHAID* algorithm

Maintainer:Jesus M. Perez <txus.perez{[at]}ehu.eus>

Class for generating a decision tree based on the CHAID* algorithm a modified version of the CHAID decision tree induction algorithm that also handles continuous features and includes the same post-pruning mechanism used by C4.5. Unlike C4.5 algorithm (J48 class), CHAID uses chi-squared, XÂ², to measure the correlation between the attributes and the class and it only handles discrete variables, although distinguishing between nominal and ordinal behaviour. New options are added to the J48 class related to the CHAID algorithm (CH). For more information, see:
