kerasZoo: Provides a wrapper classifier for zoo models available in Keras

Author:Mark Hall
Maintainer:Weka team <wekalist{[at]}list.scms.waikato.ac.nz>

Provides a wrapper classifier for invoking zoo models available in Keras. Python version 2.7.x or higher is required. Also requires the following packages to be installed in python: keras, numpy, pandas, matplotlib and scikit-learn.
